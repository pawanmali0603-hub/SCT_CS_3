# SCT_CS_3 - Password Strength Checker

## Description
This project was developed as part of the SkillCraft Technology Cyber Security Internship.

The Password Strength Checker evaluates a password based on:
- Length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

It classifies the password as Weak, Medium, or Strong and provides suggestions to improve password security.

## Features
- Checks password strength
- Gives improvement suggestions
- Easy command-line interface
- Built with Python

## Technologies Used
- Python 3
- Regular Expressions (re)

## How to Run

```bash
python password_strength_checker.py
```
